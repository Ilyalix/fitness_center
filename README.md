Приложение Фитнесс клуб.

Пользователь может ввести характеристики(вес(фактический/желаемый), рост, возраст, период, 
за который человек хочет достичь желаемого веса и приложение должно определить подходящую программу занятий.

Пользователь может быть подписан на несколько программ.

Приложение позволяет:
1) Создавать отделение фитнесс клуба(город, район города, телефоны(отдельная сущность), адрес(отдельная сущность), название, э/почта(отдельная сущность), 
возможность круглосточной работы)
2) Создавать программу занятий(кол-во кг., затрачиваемых при одноразовом посещении, название,
  рекомендуемое кол-во занятий в неделю, желаемое отделение в городе, тип программы(Very light, Light, Medium, Intensive),
  разновидность(Cross fit, Cycling, Running, Swimming, Yoga), стоимость, тренер). 
3) Хранит статистику для пользователя(фактический вес, посещение по датам), выводить сколько занятий осталось посетить для достижения желаемого веса, сколько посетил занятий. 
4) Сохранять и редактировать данные о тренере(ФИО, возраст, программы, которые может вести).
5) Редактировать программу, личную информацию тренера, информацию о клубе.
6) Удалять программу.
7) Удалять отделение, клуб.
8) Выводить информацию о клубе/отделении клуба.
9) Выводить информацию о посетителе.
10) Подписывать посетителя на акции(отправлять по почте определенный HTML шаблон).
11) Удалять посетителей, который не посещали более 90 дней.(Scheduling).

В приложении использовалось Spring 5, JPA(JPQL, NamedQueries, Criteria queries).
**** Технические требования****
Приложение:
- покрыто тестами.
- содержит четкое логическое разделение между представлениями, логикой и источником данных.
- cборка проекта/запуск тестов производится средствами maven.
- присутствует валидация данных.
